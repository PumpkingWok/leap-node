# PARSEC Labs plasma tx validation node

## Prerequisite

- Node.js 8+
- Yarn

## Install

1. `git clone git@github.com:parsec-labs/parsec-node.git && cd parsec-node`
2. `yarn`
3. `cp .config.json config.json`
